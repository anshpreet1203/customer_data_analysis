# 📊 Customer Behavior Analysis

A complete end-to-end data analysis project demonstrating data cleaning, exploratory data analysis (EDA), SQL querying, and Power BI dashboard development to uncover customer purchasing patterns and business insights.

---

# 📖 Overview

This project analyzes customer behavior using Python, SQL, and Power BI. The goal is to identify trends in customer demographics, purchasing habits, product preferences, and shopping behavior through data analysis and interactive visualizations.

The project covers:

* Data loading and preprocessing
* Exploratory Data Analysis (EDA)
* SQL-based business analysis
* Interactive Power BI dashboard creation

---

# 📂 Dataset

The dataset contains customer shopping and demographic information, including:

* Customer Category
* Age Group
* Purchase Amount
* Review Rating
* Shipping Type
* Payment Method
* Product Category
* Gender
* Location
* Discount Applied
* Previous Purchases
* Subscription Status
* Season
* Purchase Frequency
* Other customer-related attributes

The data is cleaned and transformed before analysis to ensure accurate insights.

# 🔄 Project Workflow

1. Load the dataset into Python.
2. Clean and preprocess the data.
3. Perform Exploratory Data Analysis (EDA).
4. Execute SQL queries to answer business questions.
5. Develop an interactive Power BI dashboard.
6. Generate business insights.

---

# 📈 Dashboard

The Power BI dashboard enables users to:

* Analyze customer demographics
* Track purchase trends
* Compare spending across customer categories
* Evaluate review ratings
* Analyze shipping preferences
* Explore product category performance
* Filter data interactively

---

# 💡 Key Insights

The project answers important business questions such as:

* Which customer segments generate the highest revenue?
* How does purchase amount vary across age groups?
* Which shipping methods are most popular?
* How do customer ratings differ across product categories?
* What trends can help improve business decisions?
